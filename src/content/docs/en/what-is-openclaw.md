## What is OpenClaw?

OpenClaw is an open-source AI agent framework designed for building, deploying, and managing autonomous AI agents. It provides a complete runtime environment that includes a gateway for routing, LLM engine integration, persistent memory, extensible skills via MCP, and multi-channel communication.

## Key Features

- **Gateway** — Central routing hub that manages all agent communications
- **Brain** — Flexible LLM engine supporting multiple model providers
- **Memory** — Local persistent memory system for agent context
- **Skills** — Extensible plugin system based on MCP (Model Context Protocol)
- **Channels** — Multi-platform communication (Telegram, Slack, Discord, and more)

## Why OpenClaw?

OpenClaw bridges the gap between AI capabilities and practical deployment. While tools like ChatGPT are powerful in conversation, OpenClaw turns that power into reliable, always-on agents that integrate with your existing workflows.
