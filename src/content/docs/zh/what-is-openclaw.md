## 什么是 OpenClaw？

OpenClaw 是一个开源 AI Agent 框架，专为构建、部署和管理自主 AI Agent 设计。它提供完整的运行时环境，包括路由网关、LLM 引擎集成、持久化记忆系统、基于 MCP 的可扩展技能以及多通道通信能力。

## 核心特性

- **Gateway（网关）** — 管理所有 Agent 通信的中央路由枢纽
- **Brain（大脑）** — 支持多模型提供商的灵活 LLM 引擎
- **Memory（记忆）** — 本地持久化记忆系统，维护 Agent 上下文
- **Skills（技能）** — 基于 MCP（模型上下文协议）的可扩展插件系统
- **Channels（通道）** — 多平台通信（Telegram、Slack、Discord 等）

## 为什么选择 OpenClaw？

OpenClaw 架起了 AI 能力与实际部署之间的桥梁。ChatGPT 等工具擅长对话，而 OpenClaw 将这种能力转化为可靠的、全天候运行的 Agent，无缝融入你的现有工作流。
